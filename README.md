# PHP Static CLI workflow

This workflow generates automatic binaries for my desktop/mobile PHP apps using [php-static-cli](https://github.com/crazywhalecc/static-php-cli) and is inspired by [NativePHP](https://github.com/NativePHP/php-bin).

## Using

Via Composer:

```shell
composer require par274/php-bin --dev
```
